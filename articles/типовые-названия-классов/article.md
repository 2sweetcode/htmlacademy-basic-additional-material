# Типовые названия классов для html

### Базовые
```css
.content, .main,  //основной контент
.column, .col //колонка
.footer //подвал
.head, .header //шапка, заголовок
.sidebar //сайдбар
.nav, .navigation, .navbar //навигация
.menu //меню
.map //карта
.wrapper, .wrap //обёртка
.breadcrumbs //"хлебные крошки"
.pagination //пагинация
.container //контейнер
.box, .block //блок
.row //строка
.inner //внутренний
```

### Списки
```css
.list //список
.item, .list-item //элемент списка
.unstyled //без стилей
.inline //строковый
```

### Таблицы
```css
.table //таблица
.cell //ячейка
.bordered //окаймленный
.collapsed //схлопнутый
.striped //полосатый
.condensed //сжатый
```

### Положение
```css
.bottom //низ
.center //центр
.left //лево
.right //право
.top //верх
.fixed //фиксированное
.vertical //вертикальный
.horizontal //горизонтальный
```

### Размеры
```css
.xs, .tiny, .smallest //самый маленький
.sm, .small //маленький
.md, .normal, .medium //средний
.lg, .large, .big //большой
.xl, .extra-large, .biggest //самый большой
```

### Состояние
```css
.active, .enable //активный
.deactivated, .disable //неактивный
.hidden //скрытый
.show //отображаемый
```

### Переходы
```css
.ahead //в начало
.back //назад
.forward //вперед
.next //следующий
.prev, .previous //предыдущий
.first //первый
.last //последний
```

### Контекст
```css
.active //активный
.danger //опасность
.default //по умолчанию
.error //ошибка
.info //информационный
.primary //основной
.success //успешно
.warning //внимание
```

### Действия
```css
.clear //очистка
```

### Фигуры
```css
.arrow //стрелка
.button //кнопка
```
